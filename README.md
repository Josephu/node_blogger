Blogger in Node
---------------

A good thanks to M101JS week7 mongoose course. It is good tutorial to learn MVC node webapp.

Setup
-----
1. Install node packages

    npm install

2. Setup MongoDB

    mkdir -p db/data
    mongod --dbpath db/data

3. Start node

    node app.js
